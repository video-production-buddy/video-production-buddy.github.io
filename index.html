Workflow

The agent acts like the producer, not just a clip generator.

The system is inherited from the OpenMontage architecture: pipeline manifests define what happens, director skills define how each stage should be executed, and Python tools handle concrete media work.

01

Choose the pipeline

The agent matches the brief to a production path such as animated explainer, screen demo, documentary montage, ad video, or cinematic trailer.

02

Plan before spending

Research, proposals, provider choices, render runtime options, and cost estimates are surfaced before consequential generation calls.

03

Generate with structure

Scripts, scene plans, voice, music, images, motion clips, captions, and edits are produced as auditable artifacts.

04

Review the render

Quality gates check timing, audio, black frames, subtitles, provider consistency, and whether the output matches the approved promise.

System

Built for AI coding assistants.

Instruction-first orchestration

The coding assistant reads manifests and stage skills. There is no hidden Python orchestrator making creative decisions behind the scenes.

Local and cloud media tools

Use FFmpeg, Remotion, HyperFrames, stock media, local TTS, and optional provider APIs for images, video, voice, and music.

Checkpointed production state

Each stage leaves artifacts and decisions behind, so long video jobs can be resumed and audited instead of restarted from scratch.

Examples

Several paths to a finished video.

Zero-key explainers

Render animated charts, typography, narration, and captions with local tooling and no paid media generation APIs.

Real-footage montages

Build documentary-style edits from free and open sources such as Archive.org, NASA, Wikimedia Commons, and stock providers.

Cloud-powered launches

When keys are available, route to image, video, voice, and music providers for cinematic trailers, product videos, and ads.

Quick Start

Clone, set up, then brief your agent.

Open the project in Codex, Claude Code, Cursor, Copilot, or another AI coding assistant that can read files and run commands.

shell video-production-buddy 
git clone https://github.com/zhouzhoushen/video-production-buddy.git
cd video-production-buddy
make setup

# Then ask your coding assistant:
"Make a 60-second animated explainer about how neural networks learn"

The project repository is currently private. The link and commands are ready for collaborators with access, and they will work for public visitors once the repository is made public.
